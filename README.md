# Customer Churn Prediction Using Machine Learning

## Course
Machine Learning (R1UC525B)

## Description
This repository contains a complete machine learning case study on customer churn prediction using a telecom dataset.
The project includes data preprocessing, exploratory data analysis, model training, evaluation, and comparison.

## Machine Learning Task
- Type: Binary Classification
- Target Variable: Churn

## Models Used
- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
- ROC-AUC

## Repository Structure
- Report: Final academic report (PDF)
- Presentation: PPT used for presentation
- Code: Jupyter notebook with complete implementation
- Dataset: Dataset used for training and testing
- Outputs: Graphs and plots generated during analysis
